# Time Tracker Lite

一个 Toggl 风格的轻量计时工具 MVP。

## 功能

- 开始 / 暂停 / 继续 / 停止计时
- 项目管理
- 标签与计费标记
- 条目列表与筛选
- 今日 / 本周统计
- 导出 CSV
- 数据保存在浏览器本地

## 使用

直接打开 `index.html` 即可使用。

## 本地预览

```bash
python3 -m http.server 8080
```

然后访问：`http://127.0.0.1:8080`
